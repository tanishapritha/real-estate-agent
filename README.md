# RealEstate AI Sales OS

> This repository does not contain internal build or commit guidelines.

A stateful, multi-agent workflow platform that automates the end-to-end real-estate sales lifecycle: lead intake, qualification, inventory matching, recommendations, and CRM automation. Powered by LangGraph, LiteLLM, and Model Context Protocol (MCP) servers, it drives customer engagement dynamically while maintaining complete state and observability.
